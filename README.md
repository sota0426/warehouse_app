# Rice Warehouse UI Mock

`rice_warehouse_requirements.md` をもとに、処理未接続の React Native UI モックを用意しています。

## 起動

1. `npm install`
2. `npm run web` または `npm start`

## 実装範囲

- ホーム
- 入庫
- 等級確定
- 出庫
- フレコン崩し
- 倉庫間移動
- 在庫確認
- マスタ管理
- ログイン
